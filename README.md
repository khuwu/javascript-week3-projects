# JavaScript Week 3 Projects

This repository contains the Week 3 deliverables for my Frontend Development Internship.

## Projects

### 1. Calculator

A simple calculator built using HTML, CSS, and JavaScript.

**Features**

* Addition
* Subtraction
* Multiplication
* Division
* Clear button
* Backspace button
* Responsive calculator layout

---

### 2. To-Do List

A task management application built using HTML, CSS, and JavaScript.

**Features**

* Add new tasks
* Delete tasks
* Task counter
* Clean and responsive interface

---

### 3. Number Guessing Game

A simple game where the user guesses a randomly generated number between 1 and 100.

**Features**

* Random number generation
* Feedback for high and low guesses
* Attempt counter
* Restart game functionality

---
